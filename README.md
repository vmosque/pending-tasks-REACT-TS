# 📝 Todo App - React + TypeScript

A simple Todo application built with **React** and **TypeScript**, focused on state management, type safety, and clean component structure.

---

## 🚀 Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Dynamic list rendering
- Strong typing with TypeScript

---

## 🛠️ Tech Stack

- React
- TypeScript
- Vite (or Create React App)
- CSS

---

## 🧠 Concepts Practiced

- Typed props and state in React
- Event handling with TypeScript
- Using interfaces to define data structures
- State immutability
- Component-based architecture

---

## 📌 Example Task Interface

```ts
interface Task {
  id: number;
  text: string;
  completed: boolean;
}
```

---

## 🎯 Project Purpose

This project was built as a learning exercise to reinforce how to use **TypeScript in React applications**, with a focus on writing safer and more maintainable code.

---
